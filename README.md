# 🔐 Sistema de Login e Confirmação de Senha

Projeto desenvolvido em Java para simular um sistema simples de cadastro e validação de acesso utilizando caixas de diálogo.

## Funcionalidades

* Cadastro de usuário
* Cadastro de senha
* Confirmação da senha
* Verificação dos dados informados
* Exibição de mensagens de sucesso ou erro

## Tecnologias

* Java
* JOptionPane

## Objetivo

Praticar conceitos de entrada de dados, validação de informações, estruturas condicionais e interação com o usuário utilizando JOptionPane.
